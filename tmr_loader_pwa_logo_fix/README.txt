# TMR Loader PWA - Logo Fix

Perbaikan:
- Manifest cache-bust v2
- Icon normal + maskable dipisah
- Apple touch icon ditambahkan
- Favicon ditambahkan
- Service worker cache v2

Cara pasang:
1. Upload isi folder `tmr_loader_pwa_logo_fix` ke hosting HTTPS.
2. Buka dari Chrome Android.
3. Jika sebelumnya sudah pernah install, hapus dulu app lama dari HP.
4. Di Chrome: buka Site settings > Storage > Clear data untuk domain tersebut.
5. Buka ulang URL, lalu pilih Install app / Add to Home screen.

Catatan:
- Icon PWA sering tidak berubah karena cache manifest/service worker.
- Hapus app lama dan clear site data biasanya wajib setelah ganti icon.
